Var is function scoped, and if called before it is asigned a value it will return undefined.


Let is blocked scoped, and if called before it is asigned a value it will return an error.


Const is blocked scoped, and if called before it is asigned a value it will return an error. The value also cannot be re-asigned, for example if const num=10 then num will stay 10. If you tried num=6 then you will get a type error.
